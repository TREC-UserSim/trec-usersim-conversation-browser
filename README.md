# TREC UserSim Conversation Browser

## How to use the browser?
1. Open the conversation browser: https://trec-usersim.github.io/trec-usersim-conversation-browser/
2. Obtain a recent dump of the conversation records, provided as JSONL file.
3. Load the JSONL file (use the button in the upper right corner of the browser interface).
4. Navigate through each conversation with the arrow buttons.
